# 나의 일기장 - 디자인 계획

## 앱 개요
오늘 있었던 일들을 항목별로 목록화하여 작성할 수 있는 모바일 일기장 앱.
날짜별로 일기를 관리하고, 각 일기 안에 여러 항목(이벤트/경험)을 추가할 수 있다.

---

## 화면 목록

### 1. 홈 화면 (일기 목록) - `app/(tabs)/index.tsx`
- 날짜별로 정렬된 일기 목록 표시 (최신순)
- 각 일기 카드: 날짜, 요일, 날씨 이모지(선택), 항목 개수, 첫 번째 항목 미리보기
- 상단: 앱 로고 + "나의 일기장" 제목
- 우측 상단: 새 일기 작성 버튼 (+)
- 빈 상태: 일기 없을 때 안내 메시지 + 작성 유도 버튼

### 2. 일기 작성/편집 화면 - `app/diary/[id].tsx`
- 날짜 표시 (수정 불가, 오늘 날짜 자동 설정)
- 날씨/기분 선택 (이모지 버튼: ☀️ ⛅ 🌧️ ❄️ 😊 😐 😢)
- 항목 목록 (FlatList)
  - 각 항목: 카테고리 태그 + 내용 텍스트
  - 스와이프로 삭제 또는 삭제 버튼
- 하단: "항목 추가" 버튼
- 저장 버튼 (우측 상단)

### 3. 항목 추가/편집 모달 - `components/DiaryItemModal.tsx`
- 카테고리 선택 (가로 스크롤 태그):
  - 🍽️ 식사  🏃 운동  💼 업무  📚 공부  👥 만남  🎮 여가  🏠 집안일  ✨ 기타
- 내용 입력 (멀티라인 텍스트)
- 시간 입력 (선택사항)
- 확인/취소 버튼

### 4. 일기 상세 보기 화면 - `app/diary/view/[id].tsx`
- 날짜, 날씨/기분 표시
- 항목 목록 (카테고리 태그 + 내용)
- 편집 버튼 (우측 상단)
- 뒤로가기 버튼

## 핵심 사용자 플로우

### 새 일기 작성
1. 홈 화면 → (+) 버튼 탭
2. 일기 작성 화면 열림 (오늘 날짜 자동 설정)
3. 날씨/기분 선택 (선택사항)
4. "항목 추가" 버튼 탭 → 항목 추가 모달 열림
5. 카테고리 선택 + 내용 입력 → 확인
6. 항목이 목록에 추가됨
7. 항목 여러 개 추가 가능
8. 저장 버튼 탭 → 홈 화면으로 돌아감

### 기존 일기 조회/편집
1. 홈 화면 → 일기 카드 탭
2. 일기 상세 보기 화면
3. 편집 버튼 탭 → 일기 편집 화면
4. 항목 추가/삭제/수정
5. 저장

## 데이터 구조

```typescript
interface DiaryEntry {
  id: string;
  date: string;          // YYYY-MM-DD
  weather?: string;      // 이모지 문자
  mood?: string;         // 이모지 문자
  items: DiaryItem[];
  createdAt: number;
  updatedAt: number;
}

interface DiaryItem {
  id: string;
  category: string;      // 카테고리 이름
  categoryEmoji: string; // 카테고리 이모지
  content: string;       // 항목 내용
  time?: string;         // HH:MM (선택사항)
}
```
